## Installation
### Clone Repo
```sh
git clone https://www.github.com/sudurraaa/Ansocial.git
```
```sh
cd Ansocial
```
### Install all dependencies
```sh
pip3 install -r requirements.txt
```
### Run
```sh
python3 ansocial.py
```
